    Application Guide

    This guide walks you through the four main phases of managing a project in the application: Setup & Pre-construction, Handling Changes, Construction & Monitoring, and Data Management.

    Phase 1: Setup & Pre-construction

    This phase involves setting up your project, defining its scope, calculating costs, and creating an initial schedule.

    1. Populate Your Libraries (Recommended First Step)

    Before creating a project, it's best to add common data to your libraries. This will make the cost estimation process much faster.

    • Materials Library: Go to Libraries > Materials Library to add all your common construction materials, their units, and prices.
    • DUPA Library: Go to Libraries > DUPA Labor & Equipment to define reusable resources (like "Mason" or "Excavator") and crews (pre-defined teams with specific productivity rates).

    2. Create Your Project

    Navigate to the Projects module. Click the "Add New Project" button and fill in all the relevant details like the project name, client, address, and contract information.

    3. Define the Scope in Take Off

    Go to the Take Off module and select your newly created project. Here, you'll add every work item for the project by clicking "Add Quantity". Assign each item a Category (e.g., "Earthworks," "Structural Concrete," "Finishes") for better organization.

    4. Perform Detailed Unit Price Analysis (DUPA)

    Navigate to the DUPA module and select your project. Click on a work item to define its cost breakdown by adding the required Labor, Materials, and Equipment, either manually or from your libraries.

    5. Sequence the Tasks

    Go to the Task Sequencing module. Click on a task to define its dependencies by selecting its Predecessors (tasks that must finish before) and Successors (tasks that start after).

    6. Generate Pre-construction Reports

    Go to the Pre-construction Reports module. First, generate the **Bill of Quantities (BOQ)**. This is a critical step that **locks all pre-construction data**. Once locked, you can view the **PERT-CPM**, initial **Gantt Chart**, planned **S-Curve**, and other reports.

    Phase 2: Handling Changes

    Manage Change Orders

    Navigate to the Change Orders module. You can create a new change order and add line items for new scope or to modify existing items (a Variation Order). For a change to be included in the official schedule, you must edit it and set its status to "Approved".

    Phase 3: Construction & Monitoring

    1. Set the Project Start Date

    Go to the Accomplishment Report module. The first time you file a report for a project, the app will prompt you to set the official **project start date**, which is the baseline for all tracking.

    2. File Accomplishment Reports

    In the same module, you can enter the percentage of work completed for each task on a specific date to track progress over time.

    3. Monitor Progress with Tracking Reports

    • Tracking Gantt Chart: Visually compares your planned schedule against actual progress.
    • Tracking S-Curve: Plots planned vs. actual cumulative progress to see if the project is ahead or behind schedule.
    • Revised Reports: Shows the BOQ and PERT-CPM updated to include all **approved** change orders.
    • Look-Ahead Report: Helps with short-term planning by showing upcoming tasks and their resource requirements.

    Phase 4: Data Management

    Import/Export Data

    You can easily back up and share your data. The **Projects** module has an "Export" button for individual projects. The **Libraries > Import/Export Library** module allows you to save or load your entire materials, resources, and crews library.
